# 🦀 Isopod Website

> **Minecraft Management, Encapsulated.**

Isopod is a modern, high-performance landing page for the Isopod Minecraft Server Manager. Designed with a premium, developer-centric aesthetic, it showcases the core philosophy of portability, security, and ease-of-use.

![Isopod Banner](public/Isopod_logo.png)

## ✨ Core Shell Capabilities

The Isopod ecosystem is built to redefine how modded Minecraft servers are managed:

*   **💎 Prism-Like UI**: A clean, intuitive dashboard for managing multiple isolated instances in seconds.
*   **🔄 Smart Mod Ops**: Integrated Modrinth & CurseForge search with intelligent dependency resolution.
*   **💻 Live Sync / RCON**: Real-time log streaming and interactive terminal inputs for total control.
*   **🐳 Docker Powered**: Every instance is portable, lightweight, and completely isolated from the host OS.

## 🛠 Tech Stack

The landing page reflects the same modern stack used in the core application:

*   **Frontend**: React 18 + TypeScript
*   **Styling**: Tailwind CSS (Glassmorphism & Motion)
*   **Animations**: Framer Motion
*   **Icons**: Lucide React
*   **Bundler**: Vite
